## Termux installation 
`apt update && apt upgrade`

`pkg install nodejs-lts`

`git clone https://github.com/mr-bablu/Temp-mail.git`

`cd Temp-mail`


`npm install `

`chmod +x *`

`node temp.js -u <username>`
